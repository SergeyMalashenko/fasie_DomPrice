\begin{abstract}
Отчет 5 с., 1 рис., 1 таблица.


Ключевые слова: искусственный интеллект, машинное обучение, задача регрессии, база данных


Цель выполнения НИОКР: разработка программно-аппаратного комплекса (ПАК),
предназначенного для 3D-сканирования человека и крупногабаритных объектов c
автоматической обработкой, визуализацией и подготовкой 3D-моделей для производства,
включая по технологии моделирования методом наплавления (FDM), стереолитографии
(SLA), многоструйного моделирования (MJM), ламинирования методом селективного
осаждения (SDL), струйной трехмерной печати (3DP, CJP).
Основная цель этапа — разработка основных модулей программно-аппаратного
комплекса для 3D-сканирования.
\end{abstract}
