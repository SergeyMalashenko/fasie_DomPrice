\begin{abstract}
Отчет 5 с., 1 рис., 1 таблица.


Ключевые слова: искусственный интеллект, машинное обучение, задача регрессии, база данных


Цель выполнения НИОКР: 


Разработать сервис для автоматизированной оценки жилой недвижимости и составления профессионального отчета для ипотечных сделок на базе нейросетевых технологий.
Улучшить точность нейронной сети для городов миллионников в Российской Федерации и протестировать ее работу. Усовершенствовать алгоритм обработки "сырых" данных.


Основная цель этапа — доработка существующей архитектуры сервиса, оптимизация базы данных; доработка клиентского интерфейса, доработка  модуля генерации отчетов; доработка алгоритма подбора похожих объявлений и его реализация. доработка текущих моделей оценки стоимости объектов недвижимости. разработка моделей для крупных городов.


В результате выполнения НИОКР к настоящему моменту выполнены следующие работы:
\begin{itemize}
	\item Разработан модуль, который позволяет загрузить, обработать существующий массив данных,
	\item Разработан модуль, который позволяет построить ансамбль моделей и оценить качество получаемых моделей.
	\item Разработан модуль, который позволяет оценить текущие изменения на рынке недвижимости, выполнить необходимы статистические тесты.
\end{itemize}

НИОКР по данному этапу выполнены успешно и в полном объеме.
\end{abstract}
